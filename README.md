# 彩云天气

[![hacs_badge](https://img.shields.io/badge/HACS-Default-orange.svg)](https://github.com/custom-components/hacs)

小米云服务, 支持一个账号下的多设备
提供`device_tracker`用来定位设备
后面可以提供锁机, 响铃, 云剪贴板这些服务

## 更新

+ ### v1.0
  + 支持多设备
  + 设备查找
  
## 安装配置

建议使用HACS安装和配置
